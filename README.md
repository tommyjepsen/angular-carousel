# angular-carousel
Very simple angular-js carousel.

Based on angular-js 1.3.x.

Use it like this;

carousel images="images" /carousel

scope.images = new Array();

For example;
scope.images = [
0: "image.png",
1: "image.png",
2: "image.png"];
